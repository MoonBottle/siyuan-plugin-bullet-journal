# 待办事项卡片布局调整计划

## 目标
调整 TodoSidebar.vue 中的待办事项卡片布局，从单行布局改为三行布局。

## 当前布局
```
时间 | 事项内容 | 任务名（单行）
按钮在右侧
```

## 目标布局
```
第一行：时间 | 项目名 | 按钮（靠右）
第二行：任务信息（灰色小一号字，超出换行）
第三行：事项内容（超出换行）
```

## 变更内容

### 1. HTML 结构调整
修改 `item-main` 区域，将原来的单行布局改为多行布局：

**当前结构：**
```html
<div class="item-main">
  <span class="item-time">{{ formatTime(item) || '全天' }}</span>
  <span class="item-content">{{ item.content }}</span>
  <span v-if="item.task" class="item-task">{{ item.task.name }}</span>
</div>
<div class="item-actions">...</div>
```

**目标结构：**
```html
<div class="item-header">
  <span class="item-time">{{ formatTime(item) || '全天' }}</span>
  <span class="item-project">{{ item.project?.name }}</span>
  <div class="item-actions">...</div>
</div>
<div class="item-task" v-if="item.task">{{ item.task.name }}</div>
<div class="item-content">{{ item.content }}</div>
```

### 2. 样式调整
- `.item-header`：flex 布局，时间、项目名靠左，按钮靠右
- `.item-task`：灰色小一号字，超出换行
- `.item-content`：事项内容，超出换行

### 3. 影响范围
- 待办事项卡片
- 已过期事项卡片
- 已完成事项卡片（可能需要简化，只显示时间和内容）
- 已放弃事项卡片（可能需要简化）

## 实施步骤

1. 修改待办事项卡片的 HTML 结构
2. 调整对应的 CSS 样式
3. 修改已过期事项卡片的 HTML 结构
4. 测试各种状态下的显示效果
