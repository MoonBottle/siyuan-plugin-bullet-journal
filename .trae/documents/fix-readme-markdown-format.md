# 修复 README 示例代码格式

## 问题描述

README_zh_CN.md 第52-70行的示例代码中，任务名称和任务详情之间、事项和事项详情之间有多余的空行，不符合标准 Markdown 格式。

当前格式：
```markdown
## 项目名称

> 项目描述

[需求文档](http://doc.example.com)

任务名称 #任务 @L1

[任务详情](http://doc.example.com)

事项A @2026-02-28

[事项详情](http://doc.example.com)

事项B @2026-03-01 10:00:00~12:00:00 #已完成

事项C @2026-03-01 10:00:00~12:00:00 #已放弃
```

## 修复方案

将示例代码改为标准 Markdown 格式，移除任务名称与任务详情之间、事项与事项详情之间的多余空行：

```markdown
## 项目名称
> 项目描述

[需求文档](http://doc.example.com)

任务名称 #任务 @L1
[任务详情](http://doc.example.com)

事项A @2026-02-28
[事项详情](http://doc.example.com)

事项B @2026-03-01 10:00:00~12:00:00 #已完成
事项C @2026-03-01 10:00:00~12:00:00 #已放弃
```
