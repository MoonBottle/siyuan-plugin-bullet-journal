# PomodoroActiveTimer 重新设计计划

## 需求分析

根据用户提供的截图和需求，需要对 `PomodoroActiveTimer.vue` 进行以下修改：

### 1. 取消按钮功能和结束重复，可以去掉
- 移除"取消"按钮
- "结束专注"按钮改为主要操作，但保留确认弹框

### 2. 事项信息参考 `dialog.ts`，内容丰富点
参考 `showItemDetailModal` 函数的实现，展示更详细的事项信息：
- 项目名称（如果有）
- 任务名称和层级（如果有）
- 事项内容
- 事项状态标签
- 时间信息（事项的开始时间、结束时间）
- 相关链接

### 3. 开始时间（番茄钟专注开始时间），没展示，参考图二
- 添加番茄钟专注开始时间显示
- 格式参考图二中的时间线展示方式（横向进度条或时间线样式）
- 显示格式：开始时间 ~ 预计结束时间

### 4. 重新设计布局
参考用户提供的两张截图：
- 图一：当前番茄钟界面（圆形进度条、时间显示、统计信息、操作按钮）
- 图二：时间线展示方式（显示开始时间、进度条）

## 实施步骤

### 步骤 1: 分析现有数据结构
- `ActivePomodoro` 接口包含：`itemContent`, `startTime`, `projectId`, `taskId`
- `startTime` 是番茄钟专注开始的时间戳（毫秒）
- 需要计算预计结束时间：`startTime + targetDurationMinutes * 60 * 1000`

### 步骤 2: 修改组件获取更丰富的事项信息
- 通过 `itemId` 获取完整的事项详情
- 或者扩展 `ActivePomodoro` 存储更多字段

### 步骤 3: 重新设计布局

### 步骤 4: 具体修改内容

#### Template 修改：
1. 移除取消按钮
2. 添加专注时间线区域（参考图二样式）
   - 显示开始时间
   - 显示进度条/时间线
   - 显示预计结束时间
3. 添加事项信息卡片区域，参考 `dialog.ts` 中的卡片样式
4. 调整布局结构

#### Script 修改：
1. 添加计算属性获取格式化的专注开始时间
2. 添加计算属性获取预计结束时间
3. 扩展事项信息展示（需要获取项目、任务信息）

#### Style 修改：
1. 添加专注时间线样式（参考图二的横向进度条）
2. 添加事项信息卡片样式（参考 `dialog.ts` 中的 `.sy-dialog-card` 样式）
3. 调整按钮布局（移除取消按钮后的两按钮布局）
4. 优化整体视觉层次

## 注意事项

1. **数据获取**：当前 `ActivePomodoro` 只存储了 `projectId` 和 `taskId`，需要额外获取项目/任务名称
2. **样式一致性**：保持与思源笔记主题变量一致（`--b3-theme-*`）
3. **响应式**：确保在不同尺寸下都能正常显示
4. **时间显示格式**：使用 `HH:mm` 格式显示时间
