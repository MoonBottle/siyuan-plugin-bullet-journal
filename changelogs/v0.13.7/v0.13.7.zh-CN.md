# 更新日志

## [0.13.7] - 2026-05-17

### Features

- **专注工作台**：全新专注工作台标签页、专注计划预算（番茄数/时长）、事项选择器、偏差概览统计
- **项目工作台**：重构为三栏布局（项目列表 + 任务树 + 详情面板），支持搜索、标签筛选、键盘导航
- **工作台增强**：侧边栏创建菜单新增专注工作台和项目工作台
- **桌面悬浮番茄钟**：新增独立桌面悬浮窗显示模式，支持页内/桌面/同时三种模式
- **习惯打卡增强**：新增艾宾浩斯复习频率类型

### Refactors

- **甘特图日期解析**：使用 dayjs 重构日期解析逻辑，优化边界日期处理
- **Block Writer 模块**：新增基于 blockId 的统一块写入系统，支持 API 和 Protyle 双通道写入，覆盖日期、状态、斜杠命令、置顶等操作
- **统一写入迁移**：斜杠命令、日历/甘特图事件、移动端操作、AI 工具等全面迁移至 blockWriter，废弃旧 fileUtils

### CI / Dev

- **检查脚本**：新增废弃 fileUtils 和直接 updateBlock 导入的检查脚本，集成到构建流程
