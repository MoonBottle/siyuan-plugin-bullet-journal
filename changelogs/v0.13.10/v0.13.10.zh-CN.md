# 更新日志

## [0.13.10] - 2026-06-17

### Features

- **日历视图**：支持仅事项/仅任务切换，新增事项状态筛选，可配置日期点击进入方式（单击/双击）
- **甘特图**：新增日期预设（今天/本周/本月/近7天/30天/90天/180天/全部），支持事项日期优先过滤，新增显示层级选择（仅任务/含事项），点击任务条可查看事项详情
- **待办事项**：事项卡片显示项目分组名，操作栏统一重构支持置顶/跳过/预计设置，事项详情弹框支持左右切换导航
- **工作台**：新增日历和甘特图视图组件，配置即时生效无需保存
- **番茄钟**：新增专注时长展示，右键菜单图标优化
- **AI 助手**：新增技能市场（模板市场），新增技能编辑器（支持 Markdown 编辑和预览），AI 调用出错时显示分类错误提示（认证失败/模型不可用/频率限制/网络异常），支持一键重试
- **Webhook 通知**：新增 Webhook 通知配置，支持钉钉/飞书/企微/自定义渠道，可订阅提醒、番茄钟、休息、习惯等事件
- **设置页面**：改为即时保存，新增分组切换，无需点击保存按钮；新增节假日数据管理（同步状态/覆盖范围/手动刷新）
- **图标升级**：全新自定义图标系统，提升品牌辨识度
- **MCP**：去掉 HTTP MCP 配置的"实验性"标识，全库扫描模式下自动排除未启用的目录

### Fixes

- **日历**：修复悬浮详情导致滚动条、日期选择器数字与星期不对齐、仅任务模式下事件宽度异常、切换视图配置后不立即生效等问题
- **甘特图**：修复加载时字体闪烁、多日期事项显示异常、日期范围计算少一天等问题
- **待办事项**：修复操作栏分割线残留、已放弃事项不显示完成按钮、侧边栏拖拽图标默认不收起等问题
- **重复事项**：修复时间精度丢失、分隔符从 ~ 变成 -、今日事项也显示跳过按钮等问题
- **番茄钟**：修复休息结束时系统通知未触发的问题
- **AI 助手**：修复消息不显示、技能列表不显示、设置按钮传参错误等问题
- **设置页面**：修复配置未从插件加载导致默认值覆盖、保存时机不正确等问题
- **事项详情**：修复从待办打开时导航计数不正确、项目信息未实时更新、readonly 模式下操作按钮显示异常等问题
- **移动端**：修复设置抽屉样式缺失、详情页布局异常等问题

### Internal

- **事项操作统一**：重构事项操作逻辑，统一操作入口
- **工作台视图拆分**：拆分封装独立视图组件
- **AI 技能系统**：简化技能加载和调用流程
- **块写入器重构**：优化块写入逻辑
- **斜杠命令重构**：完善范围支持
- **图标统一**：全面替换为自定义图标
