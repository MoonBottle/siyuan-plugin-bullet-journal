# Changelog

## [0.13.8] - 2026-05-19

### Features

- **Workbench View Configuration**: Support configuring group filtering, date range, sorting rules, priority/tag filtering for Todo/Quadrant/Focus/Project views; Project view supports drag-to-adjust three-pane column widths
- **Workbench AI Chat View**: New AI chat view with sidebar conversation list, search, and rename functionality
- **Date Picker Widget**: New workbench date picker supporting month/week views, Shift+click date range selection, and linkage rule configuration
- **Task Tag Parsing**: Parser now supports task-level tag parsing, task tags displayed in item detail dialog

### Refactors

- **Focus Workbench Rename**: Renamed FocusReview to FocusWorkbench to eliminate "review" semantic deviation
- **Menu & AI Store Optimization**: Optimized main menu order, AI Store now unified to read conversation list from store

### Fixes

- Fixed date picker blur, date filtering logic errors, flex layout overflow issues
- Removed Dock panel header icons, keeping only title text
