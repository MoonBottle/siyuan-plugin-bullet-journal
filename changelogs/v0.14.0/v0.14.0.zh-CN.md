# 更新日志

## [0.14.0] - 2026-07-04

### Features

- **企业微信机器人**：支持企业微信机器人集成，可通过企业微信与 AI 助手对话 ([#78](https://github.com/MoonBottle/siyuan-plugin-bullet-journal/issues/78))
  - 企微消息收发：支持在企业微信中与 AI 助手进行对话
  - 流式消息回复：支持打字机效果的流式回复
  - 通知推送：支持提醒、番茄钟等事件通过企业微信推送通知
  - 多通道分发：通知系统支持同时推送到多个渠道
  - 会话保活：支持 WebSocket 连接自动重连和会话持久化
- **CSS 隔离**：新增 scopeBytemdCss Vite 插件，隔离第三方 CSS 全局样式，避免与思源样式冲突

### Fixes

- **Skill 编辑器样式**：修复暗色模式下 skill 编辑页面的富文本编辑器和思源的样式文件冲突导致思源代码块字体颜色异常 ([#77](https://github.com/MoonBottle/siyuan-plugin-bullet-journal/issues/77))
- **企业微信机器人**：
  - 修复 pong 响应被误判为订阅响应导致 ping 指数级增长
  - 修复本地通知推送前从持久化恢复企微会话映射
  - 修复插件加载时同步加载 AI providers 配置
  - 修复启用后自动连接并持久化启用状态与通知开关
  - 修复企微消息接入会话存储使聊天记录在 UI 可见
  - 修复状态同步与单聊回复目标
  - 修正订阅响应解析匹配真实协议
  - 支持 aibot_event_callback 事件回调
  - 修正 WebSocket 端点 URL
  - 修复设置接线缺失、通知开关无效、禁用不停止监控
  - 修复移动端输入丢失、plugin 引用空值、通知开关初始化问题
  - 修复 AI 回复通道错配
- **会话选择**：修复 ConversationListItem 类型不兼容 wecom 来源
- **AI 对话**：优化企微机器人按钮点击行为
