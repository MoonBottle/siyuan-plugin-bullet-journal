# 更新日志

## [0.12.0] - 2026-04-06

### Features

- **AI 对话增强**：
  - 接入 ClawBot 对话服务
- **重复事项功能**：
  - 支持设置重复事项、完成重复事项时自动创建下一次事项
- **提醒功能**：
  - 新增提醒设置对话框
  - 显示下次提醒时间
  - 集成微信通知功能
- **番茄钟增强**：
  - 支持自动延长番茄钟设置
  - 日历日视图显示番茄钟时间块
  - 底栏进度条方向自适应计时器模式
  - 集成微信通知功能
- **表情符号标记**：用表情符号替换状态标签，支持表情符号标记
- **任务标记**：增加 📋 作为任务标记的识别
- **性能优化**：
  - 流式解析项目功能
  - SQL 批量查询优化番茄钟属性获取性能
  - 增量刷新机制优化数据存储结构
