# 更新日志

## [0.13.3] - 2026-05-08

### Features

- **待办置顶**：支持将重要事项置顶显示，使用 📌 标记置顶事项，置顶事项在待办侧栏独立分组展示
- **标签筛选**：待办侧栏新增标签筛选功能，支持多选标签聚合筛选
  - 工作台待办组件同步支持标签筛选配置
- **习惯打卡时间精度**：支持自定义打卡时间精度（日/分/秒），更精确地记录每次打卡的完成时间
  - 桌面端和移动端设置界面同步支持时间精度配置
- **项目列表优化**：
  - 分组选择器右侧添加项目名称搜索框，支持按名称和路径搜索
  - 表格视图增加文档路径列展示
  - 项目按文档路径倒序排序

### Fixes

- **移动端底栏导航**：优化底部 Tab 导航布局样式
- **移动端路由入口**：修复插件入口路由跳转到任务 Shell 的问题
