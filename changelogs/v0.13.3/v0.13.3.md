# Changelog

## [0.13.3] - 2026-05-08

### Features

- **Todo Pinning**: Support pinning important items to the top, using 📌 to mark pinned items, which are displayed in a separate group in the todo sidebar
- **Tag Filtering**: Added tag filtering to the todo sidebar, supporting multi-select tag aggregation filtering
  - Workbench todo component also supports tag filtering configuration
- **Habit Check-in Time Precision**: Support custom check-in time precision (day/minute/second) for more accurate recording of completion times
  - Desktop and mobile settings interfaces both support time precision configuration
- **Project List Optimization**:
  - Added project name search box to the right of the group selector, supporting search by name and path
  - Added document path column display in table view
  - Projects sorted by document path in descending order

### Fixes

- **Mobile Bottom Navigation**: Optimized bottom tab navigation layout styles
- **Mobile Route Entry**: Fixed plugin entry route jumping to task shell issue
