# 更新日志

## [0.11.0] - 2026-03-17

### Features

- **斜杠命令**：全新斜杠命令系统，支持快速操作任务
  - 内置命令：标记今日/明日/日期、完成任务、放弃任务、查看详情、开始专注
  - 自定义命令：支持配置自定义斜杠命令，灵活扩展工作流
- **番茄钟增强**：
  - 多行描述支持：专注计时支持输入多行事项说明
  - 专注完成弹窗：重构完成对话框，支持项目/任务/事项卡片展示
  - 底栏进度条：支持逐渐缩短显示模式
  - 自动追加日期：专注结束自动给事项追加日期
- **日期选择器增强**：
  - 周一开始：日历改为周一开始，选中日期改为圆形
- **设置面板重构**：
  - 侧边栏导航：全新设置对话框布局，左侧分类导航
  - AI Provider 交互优化：新增/编辑改为弹框交互
- **空状态引导**：
  - 任务助手空状态：添加空状态引导和示例文档创建功能
  - 专注记录空状态：添加空状态引导和示例文档创建功能
- **帮助菜单**：新增帮助菜单，支持创建示例文档，查看帮助文档
