# 更新日志

## [0.8.0] - 2026-03-08

### Features

- **AI 对话功能**：内置任务助手面板，支持 OpenAI、Kimi、DeepSeek、阶跃星辰、智谱 AI 等所有 OpenAI 兼容的供应商
  - 通过对话方式查询项目、任务和事项
  - 支持生成日报并一键插入笔记
  - 多对话管理：支持创建多个独立对话场景，对话历史自动保存
