## ✨ 四大核心亮点

### 😊 Emoji 状态标记

告别冗长标签，Emoji 让任务状态一目了然！

| 标记 | 含义 | 斜杠命令 |
|------|------|----------|
| 📋 | 任务标记 | `/rw` |
| 📅 | 日期标记 | `/jt`、`/mt`、`/rq` |
| ✅ | 已完成 | `/wc` |
| ❌ | 已放弃 | `/fq` |
| 🔁 | 重复事项 | `/cf` |
| ⏰ | 提醒设置 | `/tx` |

```markdown
# 旧格式
项目开发 #任务
项目汇报 📅2026-03-17 #已完成

# 新格式
项目开发 📋
项目汇报 📅2026-03-17 ✅
```

### 🔔 智能提醒系统

为事项设置提醒，不再错过重要任务！

- ⏰ **多种提醒方式**：支持绝对时间、相对开始时间、相对结束时间
- 📱 **微信通知**：集成 ClawBot，提醒可通过微信推送
- 🎯 **可视化设置**：事项详情弹框一键设置提醒

```markdown
周会 📅2026-03-17 14:00~16:00 ⏰提前10分钟
月度汇报 📅2026-03-17 ⏰14:00
```

### 🔄 重复事项管理

周期性任务自动创建，一次设置，持续追踪！

- 📅 **灵活重复规则**：每天、每周、每月、每年、工作日
- 📆 **指定周几**：支持每周一三五等指定周几重复
- 🔢 **结束条件**：支持按日期截止或按次数结束
- ✨ **自动创建**：标记完成时自动创建下一次事项

```markdown
每周一、三、五健身，提前5分钟提醒，坚持到今年年底 📋
健身 📅2026-03-17 18:00~19:00 ⏰提前5分钟 🔁每周一三五 截止到2026-12-31

每天背单词，早上8点提醒，坚持30天 📋
背单词 📅2026-03-17 08:00 ⏰08:00 🔁每天 剩余30

每月15号开例会，下午2点开始，提前10分钟提醒 📋
例会 📅2026-03-15 14:00~16:00 ⏰提前10分钟 🔁每月15日
```

### 🤖 ClawBot 对话集成

AI 对话能力全面升级，支持微信扫码登录！

- 💬 **微信扫码登录**：扫码即可绑定微信，开启对话
- 📱 **微信通知**：**事项提醒**、**番茄钟**可通过微信推送
- 🔄 **消息同步**：对话消息实时同步，随时查看

## 🎯 功能详解

### 提醒功能

**设置方式**：
- 事项详情弹框 → 设置提醒
- 右键菜单 → 设置提醒
- 斜杠命令 `/tx`
- 手动输入 `⏰HH:mm`

**提醒类型**：

| 类型 | 示例 | 说明 |
|------|------|------|
| 绝对时间 | `⏰14:00` | 当天 14:00 提醒 |
| 相对开始 | `⏰提前10分钟` | 事项开始前 10 分钟 |
| 相对结束 | `⏰结束前10分钟` | 事项结束前 10 分钟 |

### 重复事项

**创建方式**：
- 事项详情弹框 → 设置重复
- 右键菜单 → 设置重复
- 斜杠命令 `/cf`
- 手动输入 `🔁每天`、`🔁每周` 等

**重复规则**：

| 规则 | 示例 | 说明 |
|------|------|------|
| 每天 | `🔁每天` | 每天重复 |
| 每周 | `🔁每周` | 每周同一天 |
| 指定周几 | `🔁每周一三五` | 每周一、三、五 |
| 每月 | `🔁每月` | 每月同一天 |
| 指定日期 | `🔁每月15日` | 每月 15 日 |
| 每年 | `🔁每年` | 每年同一天 |
| 工作日 | `🔁工作日` | 工作日重复 |

**结束条件**：

| 条件 | 示例 | 说明 |
|------|------|------|
| 日期截止 | `截止到2026-12-31` | 到指定日期结束 |
| 次数限制 | `剩余10` | 剩余 10 次后结束 |

### 番茄钟增强

- ⏱️ **自动延长**：支持自动延长番茄钟设置
- 📅 **日历时间块**：日历日视图显示番茄钟时间块
- 📊 **进度条优化**：底栏进度条方向自适应计时器模式
- 💍 **圆环动画**：休息计时器圆环从满到空收缩显示

### 性能优化

- 🚀 **流式解析**：项目解析性能大幅提升
- 💾 **SQL 批量查询**：优化番茄钟属性获取性能
- 🔄 **增量刷新**：数据存储结构优化，刷新更快速
