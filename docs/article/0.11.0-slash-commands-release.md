输入 `/` 即可唤起命令面板，支持以下快捷操作：

## 🔗 斜杠命令：打通文档与视图的桥梁

任务助手始终坚持**"无侵入式"**的设计理念——文档即任务，标记即管理。斜杠命令让这一理念更加完美地落地：

### 📝 文档编辑即任务管理

在思源笔记中编辑文档时，无需切换界面，输入斜杠命令即可完成所有任务操作：

| 命令 | 操作 | 效果 |
|------|------|------|
| `/jt` | 标记今日 | 当前行添加 `@今天` 标记，自动同步到日历视图 |
| `/mt` | 标记明天 | 当前行添加 `@明天` 标记，日历视图实时更新 |
| `/rq` | 选择日期 | 弹出日期选择器，自定义任务日期 |
| `/wc` | 完成任务 | 添加完成标记，所有视图状态同步更新 |
| `/fq` | 放弃任务 | 添加放弃标记，从活跃任务中移除 |
| `/zz` | 开始专注 | 直接启动番茄钟，自动关联当前事项 |
| `/xq` | 查看详情 | 打开事项详情弹框，展示完整时间线 |
| `/rl` | 打开日历 | 自动跳转到事项所在日期，无缝切换 |
| `/rg` | 打开甘特图 | 在甘特图中定位当前任务 |

### ✨ 使用场景示例

**场景一：快速规划今日工作**
```
在每日笔记中：
- 完成项目报告 /jt          → 自动添加到今日任务
- 回复客户邮件 /jt          → 日历视图同步显示
- 代码审查 /jt              → 甘特图展示时间安排
```

**场景二：专注工作流**
```
在任务文档中：
- 撰写设计文档 /zz          → 直接启动番茄钟计时
- 专注结束                  → 自动追加专注记录到文档
- 查看统计 /rl              → 跳转到日历查看专注历史
```

**场景三：任务状态流转**
```
- 需求评审会议 /mt          → 标记为明天进行
- 会议结束 /wc              → 标记完成，视图状态更新
- 项目暂停 /fq              → 标记放弃，从活跃列表移除
```

### 💡 设计理念

> **"在笔记中加标记即可在视图中查看与管理"**

- 📄 **文档优先**：任务信息以纯文本形式保存在笔记中
- 🏷️ **标记简单**：`#任务`、`@日期`、`✓完成` 等直观标记
- 🔄 **实时同步**：文档修改自动反映到所有视图
- 🚀 **操作便捷**：斜杠命令让编辑和管理的边界消失
