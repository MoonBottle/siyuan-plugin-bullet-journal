## ✨ 斜杠命令系统重磅上线！

任务助手插件 0.11.0 版本发布，全新斜杠命令系统让任务操作更加便捷高效，告别繁琐的鼠标点击，键盘党福音来了！

***

## 🎯 斜杠命令核心功能

### 1️⃣ 内置快捷命令

输入 `/` 即可唤起命令面板，支持以下快捷操作：

- ✅ **标记日期**：快速标记今日/明日/指定日期
- ✔️ **完成任务**：一键将任务标记为已完成
- 🚫 **放弃任务**：快速放弃不再进行的任务
- 📋 **查看详情**：打开事项详情弹框
- 🍅 **开始专注**：直接启动番茄钟专注计时

<br />

## 📝 使用指南

### 快速开始

1. **更新插件**：在思源笔记插件市场中更新至 0.11.0 版本
2. **体验斜杠命令**：在任务文档中输入 `/` 唤起命令面板
3. **享受高效操作**：用键盘快速管理任务！

### 斜杠命令使用技巧

- **快速标记**：输入 `/jt `快速标记今日任务
- **一键完成**：输入 `/wc `将当前任务标记为已完成
- **开始专注**：输入 `/zz `直接启动番茄钟
