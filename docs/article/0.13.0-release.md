## ✨ 四大核心亮点

### 🎯 习惯打卡系统

全新习惯追踪功能，帮助你建立和维持日常习惯。习惯与任务并列，都是项目下的独立类型。

#### 两种习惯类型

- **二元型**：做了/没做，如「早起」「冥想」
- **计数型**：带目标值+单位，如「喝水 8杯」「跑步 5公里」

#### 习惯定义语法

```markdown
早起 🎯2026-05-01 坚持30天 ⏰07:00 🔄每天

喝水 🎯2026-05-01 坚持21天 8杯 ⏰09:00 🔄每天

跑步 🎯2026-05-01 5公里 🔄每2天
```

#### 核心功能

- 📅 **打卡记录**：在 Dock 或工作台中一键打卡，自动在笔记中创建记录
- 📊 **统计卡片**：月打卡次数、完成率、当前连续/最长连续天数
- 🔥 **打卡日历**：月视图热力图，颜色深浅表示达标程度
- 🔔 **智能提醒**：根据频率规则自动计算提醒时间，支持微信推送
- 📦 **习惯归档**：不再继续的习惯可归档，保留历史记录和统计

#### 斜杠命令

| 命令 | 功能 |
|------|------|
| `/xg` / `/habit` | 创建/编辑习惯 |
| `/dk` / `/checkin` | 快捷打卡 |
| `/xgd` / `/habits` | 打开习惯打卡 Dock |

#### 打卡方式

- **桌面端**：右侧栏 Dock → 习惯打卡
- **移动端**：底部 Tab 导航 → 习惯打卡
- **工作台**：创建习惯视图，更大展示区域

#### 补打卡

支持为历史日期补打卡，周日期行切换日期后打卡，连续打卡天数自动重新计算。

#### 习惯归档

当习惯不再继续时，在习惯详情面板点击归档，添加 `📦YYYY-MM-DD` 标记：

- 归档后停止提醒，从默认列表中隐藏
- 历史记录和统计完整保留
- 支持随时取消归档恢复

### 🛠️ 工作台（Workbench）

桌面端专属的集成工作空间，将仪表盘、习惯视图、待办视图、四象限视图和专注统计整合到同一个 Tab 中。

#### 侧边栏导航

- 左侧边栏列出所有条目（仪表盘和视图）
- 支持新建、重命名、删除条目
- 点击条目切换右侧内容

#### 仪表盘

- 网格化自由布局，支持拖拽调整位置和大小
- 5 种内置小部件自由组合：

| 小部件 | 说明 |
|--------|------|
| 待办列表 | 按预设筛选的待办事项 |
| 四象限摘要 | 按分组和象限筛选的任务摘要 |
| 习惯周视图 | 本周习惯打卡进度概览 |
| 日历视图 | 月视图日历，展示任务与习惯打卡 |
| 专注统计 | 番茄钟统计数据卡片 |

#### 视图类型

- 习惯视图、待办视图、四象限视图、专注统计视图
- 每个视图独立存在，可与仪表盘并列切换

#### 使用示例

```
「晨间仪表盘」= 今日待办 + 习惯周视图 + 专注统计
「项目仪表盘」= 四象限摘要 + 项目待办 + 专注统计
```

![晨间仪表盘](https://cdn.jsdelivr.net/gh/MoonBottle/siyuan-plugin-bullet-journal@image-host/asset/20260504202822540.png)

#### 访问方式

- 右键顶栏 → 工作台
- 顶栏下拉菜单（插件图标）→ 工作台

#### 数据持久化

所有条目、仪表盘和小部件布局自动保存，无需手动操作，重装插件后数据仍然保留。

### 📊 四象限视图

基于「重要性-紧急性」矩阵，将任务按优先级自动归类到四个象限中。

#### 象限划分

| 象限 | 名称 | 优先级 |
|------|------|--------|
| 第一象限 | 重要且紧急 | `🔥` 高优先级 |
| 第二象限 | 重要不紧急 | `🌱` 中优先级 |
| 第三象限 | 紧急不重要 | `🍃` 低优先级 |
| 第四象限 | 不重要不紧急 | 无优先级 |

#### 核心交互

- **自动归类**：根据事项上的 `🔥`/`🌱`/`🍃` 标记自动分配
- **拖拽调整**：拖拽任务到不同象限，自动更新笔记中的优先级标记
- **分组筛选**：按项目分组筛选四象限中的任务
- **搜索过滤**：关键词实时过滤四个象限的任务

![四象限视图](https://cdn.jsdelivr.net/gh/MoonBottle/siyuan-plugin-bullet-journal@image-host/asset/image%2021.png)

#### 访问方式

- 右键顶栏 → 四象限
- 顶栏下拉菜单（插件图标）→ 四象限
- 工作台中创建四象限视图

#### 每日检视建议

1. 优先处理「重要且紧急」
2. 规划时间处理「重要不紧急」，避免升级为紧急
3. 评估「紧急不重要」是否可以委托
4. 清理「不重要不紧急」中无价值的任务

## 🚀 快速开始

1. **更新插件**：在思源笔记插件市场中更新至 0.13.0 版本
2. **创建习惯**：在笔记中输入 `/xg` 创建第一个习惯
