# 🚀 Task Assistant Plugin 0.13.0 Released | Habit Check-in, Workbench, Quadrant View — Task Management Evolved!

> **Task Assistant** is a task management plugin for SiYuan Note. v0.13.0 brings three major features — **Habit Check-in**, **Workbench**, and **Quadrant View** — extending your task management from "getting things done" to "building habits", and evolving from "scattered views" to an "integrated workspace"!
>
> ⭐ **If this project helps you, please give us a Star!** [GitHub Repository](https://github.com/MoonBottle/siyuan-plugin-bullet-journal)

***

## ✨ Three Core Highlights

### 🎯 Habit Check-in System

Brand-new habit tracking to help you build and maintain daily routines. Habits are first-class citizens alongside tasks, both at the project level.

**Two Habit Types**:

- **Binary**: Done / Not done, e.g., "Wake up early", "Meditate"
- **Count**: With target value + unit, e.g., "Drink water 8 cups", "Run 5 km"

**Habit Definition Syntax**:

```markdown
Wake up early 🎯2026-05-01 Stick to 30 days ⏰07:00 🔄Every day

Drink water 🎯2026-05-01 Stick to 21 days 8 cups ⏰09:00 🔄Every day

Run 🎯2026-05-01 5 km 🔄Every 2 days
```

**Core Features**:

- 📅 **Check-in Records**: One-click check-in in Dock or Workbench, automatically creates records in notes
- 📊 **Statistics Cards**: Monthly check-ins, completion rate, current streak / longest streak
- 🔥 **Check-in Calendar**: Monthly heatmap, color depth indicates completion level
- 🔔 **Smart Reminders**: Automatically calculates reminder time based on frequency rules, supports WeChat push
- 📦 **Habit Archive**: Archive habits you no longer continue, preserving history and statistics

**Slash Commands**:

| Command | Function |
|---------|----------|
| `/xg` / `/habit` | Create / Edit habit |
| `/dk` / `/checkin` | Quick check-in |
| `/xgd` / `/habits` | Open Habit Check-in Dock |

***

### 🛠️ Workbench

A desktop-only integrated workspace that combines Dashboard, Habit View, Todo View, Quadrant View, and Focus Stats into a single Tab.

**Sidebar Navigation**:

- Left 240px sidebar listing all entries (dashboards and views)
- Supports creating, renaming, and deleting entries
- Click entries to switch right-side content

**Dashboard**:

- 12-column grid layout with drag-and-drop position and size adjustment
- 5 built-in widgets to freely combine:

| Widget | Description |
|--------|-------------|
| Todo List | Filtered todo items by preset |
| Quadrant Summary | Task summary by group and quadrant |
| Habit Week | Weekly habit check-in progress overview |
| Mini Calendar | Compact monthly calendar for item distribution |
| Focus Stats | Pomodoro statistics card |

**View Types**:

- Habit View, Todo View, Quadrant View, Focus Stats View
- Each view exists independently, switchable alongside dashboards

**Usage Examples**:

```
"Morning Dashboard" = Mini Calendar + Today's Todo + Habit Week
"Project Dashboard" = Quadrant Summary + Project Todo + Focus Stats
```

***

### 📊 Quadrant View

Based on the "Importance-Urgency" matrix, automatically categorizes tasks into four quadrants by priority.

**Quadrant Division**:

| Quadrant | Name | Priority |
|----------|------|----------|
| Q1 | Important & Urgent | `🔥` High |
| Q2 | Important & Not Urgent | `🌱` Medium |
| Q3 | Urgent & Not Important | `🍃` Low |
| Q4 | Not Important & Not Urgent | No priority |

**Core Interactions**:

- **Auto-categorization**: Automatically assigns items based on `🔥`/`🌱`/`🍃` markers
- **Drag to adjust**: Drag tasks between quadrants to automatically update priority markers in notes
- **Group filter**: Filter tasks by project group
- **Search filter**: Keyword filtering across all four quadrants

## 🎯 Feature Details

### Habit Check-in

**Check-in Methods**:

- **Desktop**: Right-click top bar → Closed Panels → Habit Check-in
- **Mobile**: Bottom Tab navigation → Habit Check-in
- **Workbench**: Create Habit View for a larger display area

**Make-up Check-ins**:

Supports backdating check-ins. Switch dates via the weekday row and check in, streak automatically recalculates.

**Habit Archive**:

When a habit is no longer continued, click Archive in the habit detail panel to add a `📦YYYY-MM-DD` marker:

- Stops reminders after archiving, hidden from default list
- History and statistics fully preserved
- Supports unarchiving at any time to restore

### Workbench

**Access**:

- Right-click top bar → Workbench
- Top bar dropdown menu (plugin icon) → Workbench

**Data Persistence**:

All entries, dashboards, and widget layouts are automatically saved to `workbench.json`. No manual operation required, data persists after reinstalling the plugin.

### Quadrant View

**Access**:

- Right-click top bar → Quadrant
- Top bar dropdown menu (plugin icon) → Quadrant
- Create Quadrant View in Workbench

**Daily Review Suggestions**:

1. Prioritize "Important & Urgent"
2. Schedule time for "Important & Not Urgent" to prevent becoming urgent
3. Evaluate whether "Urgent & Not Important" can be delegated
4. Clean up valueless tasks in "Not Important & Not Urgent"

### Mobile Adaptation

v0.13.0 includes a comprehensive mobile refactor with a **fixed bottom bar + four main tabs** unified navigation structure.

**Bottom Navigation**:

| Tab | Description |
|-----|-------------|
| **Todo** | Todo item list with pull-to-refresh |
| **Pomodoro** | Full Pomodoro timer, upgraded from drawer to independent tab |
| **Habit** | Habit list, weekday row, check-in, upgraded from todo branch to independent tab |
| **More** | Settings and other feature entries |

**Mobile Habit Check-in**:

- Tap habit card body to open a **bottom sheet** detail drawer instead of full-page navigation
- Drawer shows today's progress, statistics cards, monthly calendar, check-in log
- Supports direct check-in, backdated check-in, and count modification within the drawer
- Binary "Check-in/Checked-in" button and Count "+1" button remain on the card for quick actions without triggering details

**Mobile Pomodoro**:

- Enter Pomodoro main view directly from bottom navigation
- Supports selecting focus item, setting duration (15/25/45/60 min or custom)
- Shows circular progress bar countdown during focus
- After focus completion, choose break (5/10/15 min) or skip
- Supports both countdown and stopwatch modes

## 🚀 Quick Start

1. **Update Plugin**: Update to 0.13.0 in SiYuan Plugin Marketplace
2. **Create Habit**: Type `/xg` in notes to create your first habit
3. **Open Workbench**: Right-click top bar → Workbench, create your first dashboard
4. **Try Quadrant**: Right-click top bar → Quadrant, view task priority distribution
5. **Drag to Adjust**: Drag tasks in Quadrant to experience priority management
