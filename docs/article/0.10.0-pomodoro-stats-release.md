## 🍅 六大统计模块

### 1️⃣ 统计概览

快速了解你的专注数据概况：

- 🍅 今日番茄数及与昨日对比
- 📊 累计完成的番茄总数
- ⏱️ 今日专注时长及与昨日对比
- 📈 累计专注总时长

### 2️⃣ 年度热力图

可视化展示全年每一天的专注记录，直观了解自己的专注习惯：

- 🟢 颜色越深表示当天专注时长越长
- 📅 支持快速定位高产出日期
- 🔍 悬停查看当日专注详情

### 3️⃣ 专注占比分析

饼图展示各任务/事项的专注时长占比：

- 📊 按任务分组：了解哪些任务占用最多时间
- 📝 按事项分组：掌握具体事项的专注投入
- 🎯 发现时间分配是否合理

### 4️⃣ 专注趋势图

支持日/周/月/年多维度查看专注时长变化趋势：

- 📈 追踪专注时长变化趋势
- 📊 对比不同时期的专注表现
- 💡 发现效率提升或下降的规律
- 📉 显示日均专注时长

### 5️⃣ 周专注时段分布

横向热力图展示一周内各时间段的专注分布（7天×24小时）：

- ⏰ 了解自己在哪些时段最容易专注
- 📅 发现一周中的高效工作日
- 🌙 分析早晚时段的专注差异

### 6️⃣ 高效时段分析

智能分析一天中各时段的专注分布：

- 🏆 找出你的"黄金专注时段"
- ⏱️ 基于真实数据的时间管理建议
- 📋 为任务安排提供科学依据

## 🆕 其他新功能

### 状态栏番茄钟

新增底栏倒计时面板（可在设置中开启），无需打开侧边栏即可：

- ⏱️ 实时查看专注倒计时
- ▶️ 一键开始/暂停专注
- 🍅 快速切换专注时长

### 休息模式

专注完成后自动进入休息模式：

- ☕ 支持短休息（默认 5 分钟）和长休息（默认 15 分钟）
- 🔔 休息结束自动提醒
- ⏭️ 可跳过休息直接进入下一轮专注

### 专注完成弹窗

专注结束后弹出完成窗口，支持填写事项说明：

- 📝 记录本次专注的具体内容或心得
- 📊 查看实际专注时长
- 💾 保存说明到专注记录中

### 专注记录存储配置

支持自定义番茄钟记录存储位置：

- 📝 **事项块下（子块）**：在事项下方创建子块存储记录，便于查看历史
- 🏷️ **事项块自定义属性**：将记录存储在事项块的自定义属性中，更简洁

### AI 对话优化

- 🎨 **Markdown 渲染升级**：使用思源 Lute 引擎替代 marked.js，渲染质量大幅提升

### 设置界面重构

- 🔍 **搜索功能**：设置面板支持搜索配置项，快速找到所需设置
- 🎨 **全新布局**：使用 SySettingItem 组件统一设置项样式
- ✨ **优化交互**：AI Provider 配置校验、保存提示等细节优化

## 📝 使用指南

### 快速开始

1. **更新插件**：在思源笔记插件市场中更新至 0.10.0 版本
2. **打开统计面板**：点击「番茄钟」侧边栏右上角的图表图标
3. **查看数据**：选择时间范围，查看各类统计图表
4. **分析习惯**：根据数据调整自己的时间管理策略

### 统计面板使用技巧

- **切换视图**：点击顶部标签切换不同统计维度
- **筛选时间**：使用日期选择器或导航按钮查看特定时间段
- **分组方式**：在专注占比图中切换按任务/事项分组
- **查看详情**：悬停图表查看详细数据，点击记录跳转到笔记
