# Contributing

Thank you for your interest in Task Assistant! Feel free to submit issues for feedback or pull requests to contribute to the development.

## Development Environment

```bash
npm install    # Install dependencies
npm run dev    # Development mode (watch file changes and build)
npm run build  # Build production version
```

Build artifacts are in the `dist/` directory. You can copy this directory to SiYuan `data/plugins/siyuan-plugin-bullet-journal` for local debugging.

## How to Contribute

- **Issues & Suggestions**: Submit in [Issues](https://github.com/MoonBottle/siyuan-plugin-bullet-journal/issues)
- **Code Contributions**: Fork this repository, make changes, and submit a Pull Request
