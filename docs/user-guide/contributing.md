# 参与贡献

感谢你对任务助手的关注！欢迎通过 Issue 反馈问题、通过 Pull Request 参与开发。

## 开发环境

```bash
npm install    # 安装依赖
npm run dev    # 开发模式（监听文件变化并构建）
npm run build  # 构建生产版本
```

构建产物在 `dist/` 目录，可将该目录复制到思源 `data/plugins/siyuan-plugin-bullet-journal` 进行本地调试。

## 参与方式

- **问题与建议**：在 [Issues](https://github.com/MoonBottle/siyuan-plugin-bullet-journal/issues) 中提出
- **代码贡献**：Fork 本仓库，修改后提交 Pull Request
