# 工作台

工作台是桌面端专属的集成工作空间，将仪表盘、习惯视图、待办视图、四象限视图和专注统计视图整合到同一个 Tab 中。通过侧边栏切换不同条目，在右侧区域展示对应内容，实现一站式任务管理。

## 快速开始

### 第一步：打开工作台

- **右键顶栏** → 工作台
- **顶栏下拉菜单**（插件图标）→ 工作台

### 第二步：创建仪表盘

1. 点击侧边栏底部的「+ 新建视图」
2. 选择「新建仪表盘」
3. 点击右上角「添加组件」，选择小部件类型

### 第三步：添加视图

1. 点击「+ 新建视图」
2. 选择需要的视图类型（习惯 / 待办 / 四象限 / 专注统计）
3. 在侧边栏点击切换

## 侧边栏

工作台左侧为 240px 宽的侧边栏，分为两个区域：

### 上方（条目列表）

- 列出所有已创建的工作台条目（仪表盘和视图）
- 当前激活的条目标记高亮
- 右键条目可**重命名**或**删除**
- 点击条目切换右侧内容

### 底部（新建）

「+ 新建视图」按钮，提供 5 种创建选项：

| 选项 | 说明 |
|------|------|
| **新建仪表盘** | 创建可自定义的小部件画布 |
| **待办视图** | 嵌入待办列表 |
| **习惯视图** | 习惯工作区 |
| **四象限视图** | 重要/紧急四象限矩阵 |
| **专注统计视图** | 番茄钟统计数据 |

## 仪表盘

仪表盘是可自定义的小部件画布，使用 12 列网格布局（行高 56px，间距 16px），支持拖拽调整位置和大小。

### 添加小部件

点击仪表盘右上角的「添加组件」按钮，选择要添加的小部件类型：

| 小部件 | 图标 | 说明 | 可配置项 |
|--------|------|------|----------|
| **待办列表** | `iconList` | 按预设筛选的待办事项列表 | Todo 预设 |
| **四象限摘要** | `iconLayout` | 按分组和象限筛选的任务摘要 | 分组 + 象限 |
| **习惯周视图** | `iconCheck` | 本周习惯打卡进度概览 | 分组 + 范围（活跃/已归档） |
| **迷你日历** | `iconCalendar` | 紧凑型月历，查看事项分布 | 分组 |
| **专注统计** | `iconClock` | 番茄钟统计数据卡片 | 章节选择 |

### 小部件操作

- **拖拽移动**：拖拽标题栏左侧的 ⋮⋮ 手柄调整位置
- **调整大小**：拖拽右下角调整大小
- **菜单操作**：点击小部件右上角的 `···` 菜单
  - 重命名小部件
  - 删除小部件
  - 打开配置弹窗（支持配置的小部件）

### 布局原理

仪表盘采用 12 列网格系统：

- **总列数**：12 列
- **行高**：56px
- **间距**：16px
- **默认大小**：每个小部件默认占 6 列 × 4 行

> 工作台配置自动保存到 `workbench.json`，无需手动操作。

## 习惯视图

习惯视图（viewType: `habit`）将习惯工作区嵌入工作台，左侧为习惯列表，右侧为习惯详情面板，与习惯打卡 Dock 共享相同的数据和交互。

### 左侧：习惯列表

- 按周展示习惯列表，与 Dock 第一层一致
- 周日期行显示本周周一到周日，当天高亮
- 支持切换日期查看历史习惯状态
- 二元型显示 `已打卡` / `待打卡` + 连续天数点阵
- 计数型显示 `当前值/目标值+单位` + 进度点阵

### 右侧：习惯详情

选中习惯后展开详情面板：

- **统计卡片**：月打卡次数、总打卡次数、月完成率、当前连续/最长连续天数
- **打卡日历**：月视图，颜色深浅表示达标程度
- **打卡日志**：按时间倒序显示打卡记录
- **归档操作**：右上角可执行归档或取消归档
- **打开文档**：点击右上角「打开文档」跳转到习惯定义所在 block

详细习惯功能说明见 [习惯打卡](./habit-checkin.md)。

## 待办视图

待办视图（viewType: `todo`）将待办列表嵌入工作台，展示即将到来的任务事项。

### 显示内容

- 按时间顺序展示待办事项
- 支持按状态筛选（待办 / 已完成 / 已放弃 / 已过期）
- 点击事项可跳转到笔记中对应位置

### 与待办 Dock 的关系

待办视图与待办 Dock 共享相同的数据源和筛选逻辑，但在工作台中以更大的展示区域呈现，适合长时间关注待办清单的场景。

---

## 四象限视图

四象限视图（viewType: `quadrant`）将重要/紧急四象限矩阵嵌入工作台。

### 核心功能

- **自动归类**：根据事项上的 `🔥`/`🌱`/`🍃` 优先级标记自动分配到对应象限
- **拖拽调整**：拖拽任务到不同象限，自动更新笔记中的优先级标记
- **分组筛选**：按项目分组筛选四象限中的任务
- **搜索过滤**：关键词实时过滤四个象限的任务

详细说明见 [四象限视图](./quadrant.md)。

---

## 专注统计视图

专注统计视图（viewType: `pomodoroStats`）将番茄钟统计数据嵌入工作台。

### 显示内容

- 今日番茄数、今日专注时长
- 总番茄数、总专注时长
- 按日期分组的专注记录列表
- 点击记录可跳转到笔记中对应块

详细番茄钟功能说明见 [番茄钟](./pomodoro.md)。

## 数据持久化

工作台的条目、仪表盘和小部件布局配置自动保存到插件数据目录的 `workbench.json` 文件中。

### 保存内容

- 所有条目（仪表盘和视图）的名称和类型
- 仪表盘内小部件的类型、位置、大小和配置
- 条目在侧边栏中的顺序

### 注意事项

- 无需手动保存，所有操作自动持久化
- 数据存储在思源笔记的插件数据目录中
- 重装插件后数据仍然保留

## 常见问题

### Q: 工作台支持移动端吗？

不支持。工作台是桌面端专属功能，移动端请使用对应的 Dock（待办 Dock、习惯打卡 Dock、番茄专注 Dock）。

### Q: 仪表盘和视图有什么区别？

**仪表盘**是可自定义的小部件画布，一个仪表盘可包含多个小部件，自由拖拽布局。

**视图**是单一功能页面的嵌入（习惯 / 待办 / 四象限 / 专注统计），每个视图独立存在，不可在同一个视图中组合多个功能。

### Q: 可以创建多个仪表盘吗？

可以。通过「+ 新建视图」→「新建仪表盘」可创建多个仪表盘，每个仪表盘有独立的小部件和布局。例如：

- 「晨间仪表盘」：迷你日历 + 今日待办 + 习惯周视图
- 「项目仪表盘」：四象限摘要 + 项目待办 + 专注统计

### Q: 小部件配置会丢失吗？

不会。所有小部件的位置、大小和配置都会自动保存到 `workbench.json`。即使关闭工作台 Tab 或重启思源笔记，重新打开后布局保持不变。

### Q: 工作台的待办视图和待办 Dock 有什么区别？

两者数据源相同，但展示方式不同：

| 特性 | 待办 Dock | 工作台待办视图 |
|------|-----------|----------------|
| 展示区域 | 侧边栏（窄） | 主区域（宽） |
| 适合场景 | 快速查看 | 长时间关注 |
| 可组合性 | 独立面板 | 可与习惯/四象限等并列 |

### Q: 习惯视图和习惯打卡 Dock 有什么区别？

两者数据和交互完全一致，但工作台习惯视图在主区域展示，空间更大，适合配合其他视图一起使用。
