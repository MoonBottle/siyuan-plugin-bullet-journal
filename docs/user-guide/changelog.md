## [0.13.7] - 2026-05-17

## [0.13.0] - 2026-05-04

### 新功能

- **工作台**：自定义你的工作面板
  - 支持添加待办列表、日历、番茄钟统计、习惯周视图、四象限任务等组件

  - 按重要和紧急程度把任务分到四个区域

- **番茄钟悬浮胶囊**：全新胶囊形状设计

  - 展示倒计时、已专注时长、关联事项
  - 支持暂停/继续、结束专注、跳过休息等快捷操作

- **手机端独立界面**：为手机端设计了完整的独立操作界面
  - 底部 Tab 导航：待办、番茄钟、习惯打卡、设置四个主入口
