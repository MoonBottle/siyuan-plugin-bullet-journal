# 四象限视图

四象限视图基于「重要性-紧急性」矩阵，将任务按优先级自动归类到四个象限中，帮助您聚焦关键事项、合理安排时间。

## 快速开始

### 第一步：为任务设置优先级

在笔记中为事项添加优先级标记：

```markdown
修复线上 Bug 🔥 @2026-05-04

编写周报 🌱 @2026-05-05

整理文档 🍃 @2026-05-06
```

### 第二步：打开四象限视图

- **右键顶栏** → 四象限
- **顶栏下拉菜单**（插件图标）→ 四象限

### 第三步：拖拽调整优先级

将任务从一个象限拖拽到另一个象限，自动更新其优先级标记。

## 象限划分

四象限视图将任务按优先级分为四个面板：

| 象限 | 名称 | 对应优先级 | 说明 |
|------|------|------------|------|
| **第一象限** | 重要且紧急 | `🔥` 高优先级 | 需要立即处理的任务 |
| **第二象限** | 重要不紧急 | `🌱` 中优先级 | 需要计划安排的任务 |
| **第三象限** | 紧急不重要 | `🍃` 低优先级 | 可以考虑委托的任务 |
| **第四象限** | 不重要不紧急 | 无优先级 | 可以考虑删除的任务 |

### 分类逻辑

- 任务根据事项上的**优先级标记**自动归类
- 没有优先级标记的任务归入「不重要不紧急」象限
- 每个象限独立显示符合条件的任务列表

## 优先级标记

在笔记中使用以下标记为事项设置优先级：

| 标记 | 优先级 | 四象限归属 |
|------|--------|------------|
| `🔥` | 高 | 重要且紧急 |
| `🌱` | 中 | 重要不紧急 |
| `🍃` | 低 | 紧急不重要 |
| 无标记 | 无 | 不重要不紧急 |

### 设置方式

1. **笔记中直接标记**：在事项内容后添加 `🔥` / `🌱` / `🍃`
2. **拖拽调整**：在四象限视图中拖拽任务到不同象限，自动更新优先级

## 界面说明

四象限视图采用 2×2 网格布局：

```
┌──────────────┬──────────────┐
│  重要且紧急   │  重要不紧急   │
│    🔥        │    🌱        │
├──────────────┼──────────────┤
│  紧急不重要   │ 不重要不紧急  │
│    🍃        │   无标记      │
└──────────────┴──────────────┘
```

### 工具栏

- **分组筛选**：按项目分组筛选任务
- **搜索框**：按关键词搜索任务
- **刷新按钮**：手动刷新数据
- **更多菜单**：
  - 显示/隐藏已完成事项
  - 显示/隐藏已放弃事项
  - 显示/隐藏关联链接
  - 显示/隐藏提醒和重复标记

### 象限面板

每个象限面板显示：

- **标题**：象限名称 + 任务数量
- **任务列表**：该象限下的所有待办事项
- **点击任务**：跳转到笔记中对应位置
- **悬停预览**：悬停任务可预览详情

## 操作功能

### 拖拽调整优先级

1. 在任意象限中**按住拖拽**任务
2. 拖拽到目标象限上方，象限边框高亮
3. 松开鼠标，任务自动移动到目标象限
4. 对应事项的优先级标记自动更新

> **提示**：拖拽操作会修改笔记中的优先级标记，与直接编辑笔记效果相同。

### 分组筛选

点击工具栏的分组下拉框，按项目分组筛选四象限中显示的任务：

- **所有分组**：显示全部任务
- **指定分组**：仅显示该分组下的项目任务

### 搜索过滤

在工具栏搜索框输入关键词，实时过滤四个象限中的任务。

## 四象限摘要小部件

工作台仪表盘支持添加「四象限摘要」小部件，在仪表盘上展示指定象限的任务列表。

### 配置选项

| 配置项 | 说明 |
|--------|------|
| **分组** | 筛选指定项目分组 |
| **象限** | 选择要展示的象限（重要且紧急 / 重要不紧急 / 紧急不重要 / 不重要不紧急） |

### 使用场景

- 在「晨间仪表盘」中添加「重要且紧急」小部件，快速查看当天必须处理的事项
- 在「项目仪表盘」中添加「重要不紧急」小部件，关注长期规划任务

详细工作台说明见 [工作台](./workbench.md)。

## 使用建议

### 每日检视

建议每天开始工作时先打开四象限视图：

1. **优先处理**「重要且紧急」象限的任务
2. **规划时间**处理「重要不紧急」象限的任务，避免其升级为紧急
3. **评估**「紧急不重要」象限的任务是否可以委托
4. **清理**「不重要不紧急」象限中真正无价值的任务

### 与优先级标记的配合

四象限视图与优先级标记是同一套系统的两种表现形式：

- 在**笔记中**使用 `🔥`/`🌱`/`🍃` 标记优先级
- 在**四象限视图**中直观查看整体分布
- 通过**拖拽**快速调整优先级

### 与待办视图的区别

| 特性 | 四象限视图 | 待办视图 |
|------|------------|----------|
| 排序维度 | 重要性 × 紧急性 | 时间顺序 |
| 核心目的 | 任务优先级管理 | 任务时间管理 |
| 交互特点 | 支持拖拽调整优先级 | 按时间查看 upcoming |
| 适合场景 | 规划每日工作重点 | 查看近期待办清单 |
