# 配置说明

本文档介绍子弹笔记插件的各项配置选项。

## 目录配置

目录配置决定插件扫描哪些文档来提取任务信息。

### 添加扫描目录

1. 打开思源设置 → 插件 → 子弹笔记
2. 在「目录配置」区域点击「添加目录」
3. 输入目录路径（如 `工作安排/2026/项目`）
4. 保存设置

![目录配置截图](./images/configuration-directory.png)

### 路径格式

- 使用相对路径，相对于思源笔记本根目录
- 支持多级目录，用 `/` 分隔
- 路径不需要以 `/` 开头或结尾

### 示例

| 路径 | 说明 |
|------|------|
| `工作安排/2026/项目` | 扫描指定目录下的文档 |
| `工作安排` | 扫描工作安排及其所有子目录 |
| `日记` | 扫描日记目录 |

## 分组管理

分组功能帮助您组织和筛选项目。

### 创建分组

1. 打开插件设置
2. 在「分组管理」区域点击「添加分组」
3. 输入分组名称（如「前端项目」「后端项目」）
4. 为目录分配分组

![分组管理截图](./images/configuration-groups.png)

### 分组用途

- 在项目列表中按分组筛选
- 在甘特图中按分组查看
- 组织和归类不同类型的项目

## 午休时间配置

午休时间用于计算工作时长时扣除午休时段，影响甘特图中的任务时长显示。

### 配置方式

1. 打开插件设置
2. 找到「午休时间」设置
3. 设置开始时间和结束时间（默认 12:00-13:00）

![午休时间配置截图](./images/configuration-lunch-break.png)

### 计算规则

当事项时间范围跨越午休时段时，会自动扣除午休时间：

```
事项：09:00 ~ 14:00
午休：12:00 ~ 13:00
实际工作时长：4 小时（而非 5 小时）
```

## 常见问题

### Q: 为什么看不到任务数据？

请检查：
1. 目录路径是否与实际文件夹结构匹配
2. 项目文档是否包含 `#任务` 标记
3. 工作事项是否有有效的日期格式（`@YYYY-MM-DD`）

### Q: 如何扫描多个目录？

在目录配置中添加多个目录路径即可。每个目录可以分配到不同的分组。

### Q: 目录修改后需要重启吗？

不需要。保存设置后插件会自动重新扫描。

### Q: 如何查看扫描结果？

在项目列表视图中可以看到所有扫描到的项目和任务。
