# 视图功能

任务助手插件提供多个视图与面板，帮助您从不同角度查看和管理工作任务。

## 日历视图

以日历形式展示工作任务，支持多种视图模式。

### 访问方式

点击顶栏 📅 日历图标

### 视图模式

| 模式 | 说明 |
|------|------|
| 月视图 | 查看整月任务分布 |
| 周视图 | 查看本周详细安排 |
| 日视图 | 查看今日具体事项 |
| 列表视图 | 以列表形式展示任务 |

### 功能特性

- **任务跳转**：点击任意任务，直接跳转到笔记中对应位置
- **实时同步**：笔记中的修改自动同步到日历
- **状态显示**：已完成事项显示删除线，已放弃事项显示灰色

### 适用场景

- 日/周计划和时间安排
- 查看今日待办事项
- 回顾历史完成情况

## 甘特图

项目进度可视化，支持层级任务展示。

### 访问方式

点击顶栏 📊 甘特图图标

### 功能特性

- **层级展示**：显示任务的父子关系
- **时间跨度**：展示任务从开始到结束的时间线
- **工作时长**：计算并显示任务的工作时长（可扣除午休时间）
- **任务跳转**：点击任务跳转到笔记位置

### 适用场景

- 了解项目整体时间线
- 查看任务依赖关系
- 评估项目进度

## 项目列表

按项目分组展示任务，支持展开查看详情。

### 访问方式

点击顶栏 📁 文件夹图标

### 功能特性

- **项目分组**：按文档/分组展示项目
- **层级展开**：展开查看任务和事项详情
- **快速筛选**：按分组或状态筛选

### 适用场景

- 组织和回顾所有项目任务
- 按项目查看工作进度
- 任务归档和整理

## 待办 Dock

在侧边栏显示即将到来的待办事项，方便快速访问。

### 访问方式

右键顶栏 → 关闭的面板 → 待办事项

### 显示内容

Dock 中展示四类事项：

| 类型 | 说明 | 显示样式 |
|------|------|----------|
| **待办** | 今天及未来的未完成任务 | 正常显示 |
| **已完成** | 标记为已完成的事项 | 删除线 |
| **已放弃** | 标记为放弃的事项 | 删除线 + 灰色 |
| **已过期** | 时间过了但未完成的事项 | 红色标记 |

### 适用场景

- 快速查看今日待办
- 查看即将到来的事项
- 处理过期未完成事项

## 习惯打卡 Dock

在侧边栏追踪日常习惯，支持二元型（做了/没做）和计数型（带目标值）习惯打卡。

### 访问方式

右键顶栏 → 关闭的面板 → 习惯打卡

### 显示内容

**第一层（习惯列表）**：

- **周日期行**：显示本周周一到周日，当天高亮，点击可切换查看某天的习惯
- **习惯列表**：每个习惯一行，显示名称 + 进度/状态
  - 二元型：`已打卡` / `待打卡` + 连续天数点阵
  - 计数型：`当前值/目标值+单位` + 进度点阵
- **已达标习惯**：当期已满足频率要求的习惯标记为「本周已达标」，灰色淡化显示

**第二层（习惯详情）**：

点击习惯展开详情面板：

- **统计卡片**：月打卡次数、总打卡次数、月完成率、当前连续/最长连续天数
- **打卡日历**：月视图，颜色深浅表示达标程度（深绿=达标，浅绿=部分，灰=未打卡）
- **打卡日志**：按时间倒序显示打卡记录，计数型可快捷 +1 或修改值

### 打卡操作

- **二元型**：点击「打卡」按钮创建记录
- **计数型**：点击 `+1` 快速递增，或输入具体值
- **补打卡**：周日期行切换历史日期后可为该日期补打卡

详细说明见 [习惯打卡](./habit-checkin.md)。

## 工作台 Tab

工作台是桌面端专属的集成工作空间，将仪表盘、习惯视图、待办、四象限和专注统计整合到同一个 Tab 中。

### 访问方式

- **右键顶栏** → 工作台
- **顶栏下拉菜单**（插件图标） → 工作台

### 功能概览

| 功能 | 说明 |
|------|------|
| **侧边栏** | 左侧 240px，列出所有条目（仪表盘和视图），支持新建/重命名/删除 |
| **仪表盘** | 可自定义的小部件画布，支持 5 种小部件自由组合 |
| **习惯视图** | 嵌入习惯工作区 |
| **待办视图** | 嵌入待办列表 |
| **四象限视图** | 嵌入重要/紧急四象限矩阵 |
| **专注统计视图** | 嵌入番茄钟统计数据 |

> 工作台配置自动保存到 `workbench.json`，无需手动操作。

详细说明见 [工作台](./workbench.md)。

## 番茄专注 Dock

在侧边栏进行番茄钟计时，并查看今日/总专注统计与专注记录。

### 访问方式

右键顶栏 → 关闭的面板 → 番茄专注

### 显示内容

- **未专注时**：今日番茄数、今日专注时长、总番茄数、总专注时长四格统计，以及按日期分组的专注记录列表；点击记录可跳转到笔记中对应块。
- **专注时**：倒计时环、时间线、当前事项信息卡片，以及暂停/继续/结束专注按钮。

### 与待办的关系

可从待办侧栏、日历或甘特图中对某条事项点击「开始专注」，也可在番茄专注 Dock 内点击播放按钮选择待办后开始。专注完成后，记录会写入该事项块下方。

详细说明见 [番茄钟](./pomodoro.md)。

## 视图对比

| 视图 | 维度 | 适用场景 |
|------|------|----------|
| 日历 | 时间 | 日/周计划，时间安排 |
| 甘特图 | 项目进度 | 时间线，任务依赖 |
| 项目列表 | 项目组织 | 任务归类，项目回顾 |
| 待办 Dock | 快速访问 | 今日事项，过期提醒 |
| 习惯打卡 Dock | 习惯追踪 | 日常习惯打卡，连续天数，完成率 |
| 工作台 Tab | 集成工作空间 | 仪表盘 + 习惯 + 待办 + 四象限 + 专注统计 |
| 番茄 Dock | 专注与记录 | 番茄钟计时，今日/总统计，记录列表 |

## 双向链接

所有视图都支持双向链接功能：

- 点击任务 → 跳转到笔记中对应位置
- 在笔记中修改 → 自动同步到所有视图
- 每个任务都有唯一的持久化链接

这是通过思源笔记的 Kramdown 格式实现的，每个段落都有唯一的 `blockId`。

## 在文档中查看事项详情

编辑项目文档时，可直接在笔记中打开事项详情弹框，无需切换到日历或待办视图。

### 操作方式

| 方式 | 操作 | 说明 |
|------|------|------|
| **右键菜单** | 在事项块上右键 → 选择菜单项 | 可发现性高，不干扰编辑 |
| **快捷键** | 按住 Ctrl（Windows）或 Cmd（Mac）点击事项内容 | 一步打开事项详情，适合熟悉用户 |

**右键菜单选项**：

- **查看详情**：打开事项详情弹框
- **在日历中查看**：打开日历视图并定位到事项日期

### 弹框内容

事项详情弹框展示：

- 所属项目和任务
- 时间、时长、专注总时长
- 事项状态（待办/已完成/已放弃/已过期）
- 关联链接
- 操作按钮：打开文档、在日历中查看

### 使用前提

- 文档需在任务助手的配置目录内
- 需先打开过日历、待办或项目视图，以便插件加载数据
