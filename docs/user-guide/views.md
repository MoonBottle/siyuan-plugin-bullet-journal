# 视图功能

子弹笔记插件提供四个主要视图，帮助您从不同角度查看和管理工作任务。

## 日历视图

以日历形式展示工作任务，支持多种视图模式。

### 访问方式

点击顶栏 📅 日历图标

### 视图模式

| 模式 | 说明 |
|------|------|
| 月视图 | 查看整月任务分布 |
| 周视图 | 查看本周详细安排 |
| 日视图 | 查看今日具体事项 |
| 列表视图 | 以列表形式展示任务 |

### 功能特性

- **任务跳转**：点击任意任务，直接跳转到笔记中对应位置
- **实时同步**：笔记中的修改自动同步到日历
- **状态显示**：已完成事项显示删除线，已放弃事项显示灰色

### 适用场景

- 日/周计划和时间安排
- 查看今日待办事项
- 回顾历史完成情况

## 甘特图

项目进度可视化，支持层级任务展示。

### 访问方式

点击顶栏 📊 甘特图图标

### 功能特性

- **层级展示**：显示任务的父子关系
- **时间跨度**：展示任务从开始到结束的时间线
- **工作时长**：计算并显示任务的工作时长（可扣除午休时间）
- **任务跳转**：点击任务跳转到笔记位置

### 适用场景

- 了解项目整体时间线
- 查看任务依赖关系
- 评估项目进度

## 项目列表

按项目分组展示任务，支持展开查看详情。

### 访问方式

点击顶栏 📁 文件夹图标

### 功能特性

- **项目分组**：按文档/分组展示项目
- **层级展开**：展开查看任务和事项详情
- **快速筛选**：按分组或状态筛选

### 适用场景

- 组织和回顾所有项目任务
- 按项目查看工作进度
- 任务归档和整理

## 待办 Dock

在侧边栏显示即将到来的待办事项，方便快速访问。

### 访问方式

右键顶栏 → 关闭的面板 → 待办事项

### 显示内容

Dock 中展示四类事项：

| 类型 | 说明 | 显示样式 |
|------|------|----------|
| **待办** | 今天及未来的未完成任务 | 正常显示 |
| **已完成** | 标记为已完成的事项 | 删除线 |
| **已放弃** | 标记为放弃的事项 | 删除线 + 灰色 |
| **已过期** | 时间过了但未完成的事项 | 红色标记 |

### 适用场景

- 快速查看今日待办
- 查看即将到来的事项
- 处理过期未完成事项

## 视图对比

| 视图 | 维度 | 适用场景 |
|------|------|----------|
| 日历 | 时间 | 日/周计划，时间安排 |
| 甘特图 | 项目进度 | 时间线，任务依赖 |
| 项目列表 | 项目组织 | 任务归类，项目回顾 |
| 待办 Dock | 快速访问 | 今日事项，过期提醒 |

## 双向链接

所有视图都支持双向链接功能：

- 点击任务 → 跳转到笔记中对应位置
- 在笔记中修改 → 自动同步到所有视图
- 每个任务都有唯一的持久化链接

这是通过思源笔记的 Kramdown 格式实现的，每个段落都有唯一的 `blockId`。
